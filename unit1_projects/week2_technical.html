What are the differences between relative, absolute, and fixed positioning?

What is positioning?

When making a web page, you using text, specifically html, and styles using CSS into a web page. There are sections, sidebars, etc. and they may be laid out in infinite ways. The HTML, which makes up the contents of the page, is basically a list of elements in the page, like lines in a book. On a website you don't simply want a series of elements from the top to the bottom of your page! You want to arrange the page to your liking. In order to do that, you need to use positioning.

Every element on the page is a block, or essentially a rectangular space with pixels in it. That block can be positioned in a number of ways, including by designating the position as relative, absolute, and fixed.

Relative

The key to understanding relative positioning is understanding that it means 'relative to itself.' If you add position:relative to a block, that alone does nothing. But if you add something like top:10px, now you are moving the element 12 px down from where it was.

Absolute

Absolute positioning allows you to position an item exactly where you want it to go on the page. For example h1{ position: absolute; left: 150px; top: 10px } will position your h1 heading 150px from the left of the page and 10px down from the top.

Fixed

Fixed positioning makes the element stay where it is in the browser regardless of if you scroll up or down. This is often used on heading with links that people might want to access from anywhere on the page.
