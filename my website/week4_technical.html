Enumerable#map

You have a list of scores out of 100 in an array and you need to do some mathamatics on them. To do the math you need to convert them to decimals. You also want to use the scores in sentences in an email to tell your friends. To do this you need to convert all of the scores to strings. How do you do it? You can do it with the map function. 


        scores = [99, 100, 97, 65, 105, 0, 92, 100]
        
        scores.map(&:to_s)
         => ["99", "100", "97", "65", "105", "0", "92", "100"] 

        scores.map {|s| 0.01*s }
         => [0.99, 1.0, 0.97, 0.65, 1.05, 0.0, 0.92, 1.0]

The map function iterates through an array and creates a new array with it! It can do things the items in the array at the same time. The first call of scores.map above is creating a new array with all elements converted to a string.

The second call of scores.map is creating a new array with all elements multiplied by .01. The scores are converted to decimals!

Pretty impressive for one line of code!
