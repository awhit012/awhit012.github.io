Learn, Lok Up, Play

Resource page

Over the last nine weeks, and on my own before I started this journey to DBC, I have come accross amazing resources on the web for learning, practicing and experimenting with programming and web development technologies. Here I've grouped some of my favorites into Learn, Look Up, and Play.

Learn

  • Udacity is an amazing site with both free and paid classes. Their CS 101 course in Python is how I got started programming. Learn everything from Java Script to A.I. for robotics.
  • Code Academy is great for introducing concepts and practicing syntax.
  • CS 50 lectures from Harvard. A charismatic instructor who is good at explaining things covers a lot.
  • Ruby Monk is my favorite way to practice Ruby. Beautiful web design, creative challenges, lighthearted tone.
  • Rails Tutorial by Michael Hartl is the definitive tutorial for Ruby on Rails.

Look Up

  • Here are 75 quick lookup cheatsheets. I could link to individual ones that I have used, but they already did a great job. HTML, CSS, JS, Ruby, SQLite, its all there and much more.
  • While I'm at it resrc.io is an incredible crowd-sourced dynamic list of free programming resources with quality rankings by the community!

Play

  • Code Avengers is a game where you direct your character using Java Script.
  • Wavepot is an in-browser music generator in Java Script. It is the most fun I've had with JS yet!
